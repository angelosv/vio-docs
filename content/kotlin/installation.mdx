# Installation

Add the Vio Kotlin SDK to your Android project using Gradle.

## Requirements

| Platform | Minimum |
|----------|---------|
| Android API | 24+ (Android 7.0) |
| Kotlin | 1.9+ |
| Jetpack Compose BOM | 2024.09.01+ |
| Android Studio | Hedgehog (2023.1.1)+ |

## Gradle

The SDK is published on **Maven Central** as `live.vio:vio-kotlin-sdk`.

### `build.gradle.kts` (Kotlin DSL)

```kotlin
dependencies {
    implementation("live.vio:vio-kotlin-sdk:1.0.1")
}
```

Make sure `mavenCentral()` is in your repository list:

```kotlin
repositories {
    mavenCentral()
    google()
}
```

### `build.gradle` (Groovy)

```groovy
dependencies {
    implementation "live.vio:vio-kotlin-sdk:1.0.1"
}
```

## Permissions

Add to your `AndroidManifest.xml`:

```xml
<uses-permission android:name="android.permission.INTERNET" />
```

## Modules

| Module | Description |
|--------|-------------|
| `VioCore` | Required — configuration, CampaignManager, models |
| `VioEngagementSystem` | Polls, contests, WebSocket engagement |
| `VioEngagementUI` | Engagement Composable components |
| `VioCastingUI` | Sports casting overlays |
