# Installation

Add the Vio Kotlin SDK to your Android project using Gradle.

## Gradle (Recommended)

The SDK is published on **Maven Central**.

### Using `build.gradle.kts` (Kotlin DSL)

Add the dependency to your app-level `build.gradle.kts`:

```kotlin
dependencies {
    implementation("io.github.reachudevteam:reachu-kotlin-sdk:1.0.1")
}
```

Make sure `mavenCentral()` is included in your repositories block (usually in `settings.gradle.kts`):

```kotlin
dependencyResolutionManagement {
    repositories {
        google()
        mavenCentral()
    }
}
```

### Using `build.gradle` (Groovy DSL)

```groovy
dependencies {
    implementation 'io.github.reachudevteam:reachu-kotlin-sdk:1.0.1'
}
```

## Jetpack Compose Setup

The SDK requires Jetpack Compose. Enable it in your app-level `build.gradle.kts`:

```kotlin
android {
    buildFeatures {
        compose = true
    }
    composeOptions {
        kotlinCompilerExtensionVersion = "1.5.14"
    }
}

dependencies {
    val composeBom = platform("androidx.compose:compose-bom:2024.09.01")
    implementation(composeBom)
    implementation("androidx.compose.ui:ui")
    implementation("androidx.compose.material3:material3")
    implementation("androidx.lifecycle:lifecycle-runtime-ktx:2.7.0")

    // SDK
    implementation("io.github.reachudevteam:reachu-kotlin-sdk:1.0.1")
}
```

## Internet Permission

Add the internet permission to your `AndroidManifest.xml`:

```xml
<uses-permission android:name="android.permission.INTERNET" />
```

## Minimum Requirements

| Setting | Minimum Value |
|---------|--------------|
| Android API | 24 (Android 7.0) |
| Kotlin | 1.9 |
| Compose BOM | 2024.09.01 |
| Java | 17 |
| Gradle | 8.0+ |
