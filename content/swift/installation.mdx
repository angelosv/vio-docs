# Installation

Add the Vio Swift SDK to your project using Swift Package Manager or CocoaPods.

## Swift Package Manager (Recommended)

### Using Xcode

1. Open your Xcode project
2. Go to **File → Add Package Dependencies...**
3. Enter the repository URL:

   ```
   https://github.com/angelosv/VioSwiftSDK.git
   ```

4. Select the version (e.g. `1.0.0` or latest)
5. Choose the modules you need:
   - **VioCore** — Required
   - **VioUI** — SwiftUI components
   - **VioLiveShow** — Livestream logic
   - **VioLiveUI** — Livestream UI
   - **VioComplete** — All modules

### Using Package.swift

Add to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/angelosv/VioSwiftSDK.git", from: "1.0.0")
]
```

Then add the products to your target:

```swift
.target(
    name: "YourApp",
    dependencies: [
        .product(name: "VioUI", package: "VioSwiftSDK"),
        .product(name: "VioCore", package: "VioSwiftSDK")
    ]
)
```

## CocoaPods

Add to your `Podfile`:

```ruby
platform :ios, '15.0'

target 'YourApp' do
  use_frameworks!
  
  pod 'VioSwiftSDK', :git => 'https://github.com/angelosv/VioSwiftSDK.git', :tag => '1.0.0'
end
```

Then run:

```bash
pod install
```
