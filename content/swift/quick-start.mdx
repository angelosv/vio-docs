# Quick Start

Get the Vio Swift SDK running in your app in under 5 minutes.

## 1. Add the Package

In Xcode: **File → Add Package Dependencies...**

```
https://github.com/angelosv/VioSwiftSDK.git
```

Select the latest version and add the modules you need (e.g. `VioCore`, `VioUI`).

## 2. Configure the SDK

In your app's entry point (e.g. `@main` App struct):

```swift
import VioCore

@main
struct MyApp: App {
    init() {
        // Configure with your API credentials
        ConfigurationLoader.loadConfiguration()
        // Or: VioConfiguration.configure(apiKey: "your-api-key", environment: .production)
    }
    
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}
```

## 3. Add a Product Card

```swift
import SwiftUI
import VioUI

struct ProductListView: View {
    let product: Product // Your product model from VioCore
    
    var body: some View {
        VStack {
            RProductCard(product: product, onAddToCart: { })
        }
    }
}
```

## 4. Run Your App

Build and run. You should see the product card rendered with the Vio styling.
