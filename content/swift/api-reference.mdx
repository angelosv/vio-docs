# API Reference

Reference for public APIs in VioCore, VioEngagementSystem, VioUI, and VioCastingUI.

## VioCore

### VioConfiguration

Global configuration singleton. Call `configure()` once at startup.

```swift
import VioCore

// Access state
let config = VioConfiguration.shared
let isConfigured = config.isConfigured
let isAvailable = config.isMarketAvailable
let shouldUse = config.shouldUseSDK  // isConfigured && isMarketAvailable

// Runtime updates
VioConfiguration.updateTheme(.dark)
VioConfiguration.updateCartConfiguration(CartConfiguration(...))
```

### ConfigurationLoader

Load config from bundle, string, remote URL, or environment variables.

```swift
import VioCore

// Recommended: auto-detect vio-config.json
ConfigurationLoader.loadConfiguration(userCountryCode: "NO")

// Specific file
ConfigurationLoader.loadConfiguration(fileName: "vio-config-viaplay", userCountryCode: "NO")

// From JSON string
try? ConfigurationLoader.loadFromJSONString(jsonString)

// From remote URL
try await ConfigurationLoader.loadFromRemote(url: URL(string: "https://...")!)

// From environment variables (CI / Xcode Scheme)
ConfigurationLoader.loadFromEnvironment()
```

### CampaignManager

Manages campaign discovery, WebSocket connection, and component lifecycle.

```swift
import VioCore

// Connect to a broadcast
let context = BroadcastContext(
    broadcastId: "real-madrid-vs-barcelona-2025-01-24",
    broadcastName: "Real Madrid vs Barcelona"
)
await CampaignManager.shared.setBroadcastContext(context)

// Read state
let campaign = CampaignManager.shared.currentCampaign
let context = CampaignManager.shared.currentBroadcastContext
let isConnected = CampaignManager.shared.isConnected
let isActive = CampaignManager.shared.isCampaignActive

// Disconnect
CampaignManager.shared.disconnect()

// Manual discovery
await CampaignManager.shared.discoverCampaigns(broadcastId: "broadcast-id")
```

### BroadcastContext

Links the SDK to a broadcast configured in the Dashboard.

```swift
public struct BroadcastContext: Codable, Equatable {
    public let broadcastId: String       // Required — matches externalId in Dashboard
    public let broadcastName: String?    // Optional display name
    public let startTime: String?        // ISO 8601
    public let channelId: Int?
    public let metadata: [String: String]?
}
```

### Campaign

```swift
public struct Campaign: Codable, Identifiable {
    public let id: Int
    public let startDate: String?
    public let endDate: String?
    public let isPaused: Bool?
    public let campaignLogo: String?
    public let sponsor: SponsorInfo?
    public let broadcastContext: BroadcastContext?
}
```

### SponsorInfo

Sponsor identity delivered from the backend. No hardcoding needed.

```swift
public struct SponsorInfo {
    public let name: String?
    public let logoUrl: String?
    public let avatarUrl: String?
    public let primaryColor: String?     // e.g. "#F5153B"
    public let badgeText: String?
}
```

### CampaignConfiguration

Set in `vio-config.json` under `"campaigns"`.

```swift
CampaignConfiguration(
    webSocketBaseURL: "wss://api.vio.live",
    restAPIBaseURL: "https://api.vio.live",
    campaignApiKey: "YOUR_CAMPAIGN_API_KEY",
    autoDiscover: true
)
```

## VioEngagementSystem

### EngagementManager

Manages poll and contest state. Updated automatically via WebSocket.

```swift
import VioEngagementSystem

@StateObject var engagement = EngagementManager.shared

// Current polls for active broadcast
engagement.activePollsForCurrentBroadcast

// Current contests
engagement.activeContestsForCurrentBroadcast

// Vote on a poll
await engagement.vote(pollId: poll.id, optionId: optionId)
```

## VioUI

### ProductService

Load products from the Commerce API.

```swift
import VioUI

// Single product
let product = try await ProductService.shared.loadProduct(
    productId: "408841",
    currency: "NOK",
    country: "NO"
)

// Multiple products
let products = try await ProductService.shared.loadProducts(
    productIds: [408841, 408895],
    currency: "NOK",
    country: "NO"
)

// Clear cache
ProductService.shared.clearCache()
```

### CartManager

```swift
import VioUI

@StateObject private var cartManager = CartManager()

await cartManager.addProduct(product)
cartManager.isCheckoutPresented = true

let count = cartManager.cart?.lineItems.count ?? 0
```
