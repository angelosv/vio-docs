# Vio Documentation

Documentation site for Vio SDK built with [Nextra](https://nextra.site) and the Docs theme.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm start
```

## Deploy

This project is configured for deployment on [Vercel](https://vercel.com). Connect your GitHub repository and deploy.

Domain: **docs.vio.live**
